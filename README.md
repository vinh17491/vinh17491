## 🌙 About Me

I am a **Software Engineering student at HUFLIT** who is especially interested in backend development, system design, and practical software architecture.

My main direction is building applications with clear business logic, maintainable source code, reliable data flows, secure authentication, and APIs that can be tested and extended without unnecessary complexity.

Although backend engineering is my primary focus, I also develop complete applications across web, mobile, and desktop platforms. This helps me understand how user interfaces, application services, databases, and infrastructure work together as one system.

I value:

- Clear and maintainable architecture
- Reliable business workflows
- Secure authentication and authorization
- Readable, testable, and reusable code
- Continuous learning through practical projects

> 💡 **Engineering mindset**  
> *“Build systems that are easy to understand, safe to change, and ready to grow.”*
