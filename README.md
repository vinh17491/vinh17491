## 💫 About Me

I am a **Software Engineering student at HUFLIT** with a strong interest in backend engineering, system design, and building applications that solve practical problems.

I enjoy transforming real requirements into clear and maintainable software systems with well-structured APIs, reliable data flows, secure authentication, and interfaces that are simple to use.

My current goal is to become a well-rounded **full-stack developer** while continuing to strengthen my backend development skills. I value readable code, careful testing, clear architecture, and solutions that remain stable as a project grows.
